Scribus on Haiku

Thanks to the efforts of Giovanni Mugnai Scribus is also available for Haiku.

To run Scribus on Haiku you need to download Scribus for Haiku, which comes in a ZIP archive. Extract the archive to Haiku’s /boot directory. The archive includes all dependencies. It also contains the latest versions of Ghostscript and LittleCMS, so these need not be installed separately.

After extraction, an icon/link to Scribus will be present on the Desktop, and you can launch Scribus.

Caveats

  • Since CUPS hasn’t been ported to Haiku yet, you can’t print directly from Scribus. You can, however, export a PDF and print from BePDF or DocumentViewer.
  • The Scripter doesn’t work, because Python Tkinter is currently not available for Haiku.
